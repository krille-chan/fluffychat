# FluffyChat ஐ நான் எவ்வாறு ஆதரிக்க முடியும்?

FluffyChatக்கு உங்கள் உதவி தேவை!

FluffyChat தன்னார்வலர்களால் உருவாக்கப்பட்டது மற்றும் அனைவருக்கும் எப்போதும் இலவசமாக இருக்கும். ஆனால் நிதி உதவி இல்லாமல் ஹோஸ்டிங் மற்றும் மேம்பாடு சாத்தியமில்லை.

இப்போது நான் கிட்டத்தட்ட எனது ஓய்வு நேரத்தை FluffyChat ஐ உருவாக்குவதற்கு செலவழிக்கிறேன். இருப்பினும், இது என்னால் எப்போதும் செய்யக்கூடிய ஒன்றல்ல. FluffyChat இன் எதிர்காலம் உங்களைப் போன்றவர்களைச் சார்ந்திருக்கிறது.

பெரும்பாலானவர்கள் இந்தச் செய்தியைப் புறக்கணிப்பார்கள். FluffyChat உங்களுக்குப் பயனுள்ளதாக இருந்தால், ஒரு சிறிய நன்கொடையை வழங்கவும்.

# FluffyChat என்றால் என்ன?

FluffyChat என்பது Matrix protocolஐ அடிப்படையாகக் கொண்ட பயனர் நட்பு, இலவசம் மற்றும் விளம்பரமில்லாத திறந்த மூலப் பயன்பாடாகும், மேலும் இது Telegram அல்லது WhatsApp போன்று செயல்படுகிறது.
FluffyChat மூலம், நீங்கள் உங்கள் நண்பர்களுடன் தனிப்பட்ட முறையில் அரட்டையடிக்கலாம், குழு அரட்டைகளை உருவாக்கலாம், உங்கள் இருப்பிடத்தைப் பகிரலாம், குரல் செய்திகளை அனுப்பலாம். சிறந்த பகுதி:
FluffyChat மற்ற மேட்ரிக்ச் தூதர்களுடன் இணக்கமானது!

# FluffyChat யாருக்காக?

FluffyChat என்பது WhatsApp, Telegram அல்லது WeChat போன்ற மையப்படுத்தப்பட்ட செய்தியிடல் பயன்பாடுகளுக்கு வேகமான, பயனர் நட்பு மற்றும் பாதுகாப்பான மாற்றாகத் தேடுபவர்களுக்கானது.

# FluffyChat ஐ உருவாக்கியது யார்?

FluffyChat சமூகத்தால் உருவாக்கப்பட்டது மற்றும் கிறிச்டியன் (Krille) Kußowski ஆல் தொடங்கப்பட்டது. இருப்பினும், குறியீடு அல்லது மொழிபெயர்ப்புகளில் நிறைய பங்களிப்பாளர்கள் உள்ளனர். அவற்றை இங்கே காணலாம்.

# FluffyChat எவ்வாறு நிதியளிக்கப்படுகிறது?

FluffyChat என்பது தன்னார்வலர்களால் இயக்கப்படும் ஒரு இலாப நோக்கற்ற திட்டமாகும், மேலும் இது நன்கொடைகளால் முழுமையாக நிதியளிக்கப்படுகிறது.

# எனது சாதனத்தில் FluffyChat கிடைக்குமா?

ஆம்! மொபைல் போன்கள், டேப்லெட்டுகள் மற்றும் உங்கள் கணினியில் இணைய பதிப்பாக FluffyChat ஐப் பயன்படுத்தலாம். ஆண்ட்ராய்டு (ப்ளே கடை மற்றும் எஃப்-டிராய்டு), ஐஇமு (ஆப் ச்டோர்), வலைப் பயன்பாடாகவும், லினக்சிற்கான டெச்க்டாப் பயன்பாடாகவும் (Flathub, Snap Store) கிடைக்கிறது. Flatpak/Snap இன் Arm64 பதிப்பைப் பயன்படுத்தி லினக்ச் ஃபோன்களிலும் இது வேலை செய்கிறது. நீங்கள் விரும்பினால் அதை MacOS மற்றும் சாளரங்கள் க்காகத் தொகுக்கலாம், ஆனால் அது இன்னும் ஆதரிக்கப்படவில்லை.

# FluffyChat கணக்கை எப்படி உருவாக்குவது?

நீங்கள் முதலில் பயன்பாட்டைத் தொடங்கும்போது, புதிய அணி கணக்கை உருவாக்கும்படி கேட்கப்படுவீர்கள். சேவையகத்தைத் தேர்ந்தெடுக்கவும் (எ.கா., matrix.org), நீங்கள் விரும்பும் பயனர்பெயர் மற்றும் பாதுகாப்பான கடவுச்சொல்லை உள்ளிடவும். விருப்பமாக, நீங்கள் மின்னஞ்சல் முகவரியைச் சேர்க்கலாம் (கடவுச்சொல் மீட்டெடுப்பிற்கு இது உதவியாக இருக்கும், ஆனால் தேவையில்லை).

# மேட்ரிக்ச் அடையாளம் என்றால் என்ன?

உங்கள் மேட்ரிக்ச் ஐடி என்பது நெட்வொர்க்கில் உள்ள உங்களின் தனிப்பட்ட பயனர்பெயர், எ.கா., @name:matrix.org. இது உங்கள் பெயர் மற்றும் உங்கள் கணக்கு அமைந்துள்ள சேவையகத்தைக் கொண்டுள்ளது.

# FluffyChat இல் நான் எவ்வாறு உள்நுழைவது?

உங்களிடம் ஏற்கனவே அணி கணக்கு இருந்தால், உங்கள் அணி அடையாளம் (எ.கா. @name:matrix.org) மற்றும் கடவுச்சொல்லை உள்ளிடவும். FluffyChat பரவலாக்கப்பட்டதால் நீங்கள் எந்த மேட்ரிக்ச்-இணக்கமான சர்வரிலும் உள்நுழையலாம்.

# FluffyChatக்கு எனக்கு மின்னஞ்சல் முகவரி தேவையா?

இல்லை, மின்னஞ்சல் முகவரி தேவையில்லை. இருப்பினும், உங்கள் கடவுச்சொல்லை மீட்டமைக்க அல்லது உங்கள் கணக்கைச் சரிபார்க்க விரும்பினால் பரிந்துரைக்கப்படுகிறது.

# எனது கடவுச்சொல்லை மறந்துவிட்டேன். இப்போது என்ன?

உங்கள் கணக்கை உருவாக்கும் போது மின்னஞ்சல் முகவரியை வழங்கியிருந்தால், சேவையக இணையதளம் (எ.கா., https://matrix.org) வழியாக உங்கள் கடவுச்சொல்லை மீட்டமைக்கலாம். எதிர்பாராதவிதமாக, மின்னஞ்சல் முகவரி இல்லாமல் மீட்பு சாத்தியமில்லை. இந்த வழக்கில், நீங்கள் ஒரு புதிய கணக்கை உருவாக்க வேண்டும்.

# நான் எனது சாதனத்தை மாற்றினேன், இப்போது எனது எல்லா செய்திகளும் குறியாக்கம் செய்யப்பட்டுள்ளன. இப்போது என்ன?

புதிய சாதனத்தில் உங்கள் மறைகுறியாக்கப்பட்ட செய்திகளைப் படிக்க, நீங்கள் முன்பு உருவாக்கிய காப்புப்பிரதியை மீட்டெடுக்க வேண்டும். புதிய சாதனத்தில் FluffyChat ஐ அமைக்கும் போது உங்கள் காப்பு விசை அல்லது கடவுச்சொல்லை உள்ளிடவும். அப்போதுதான் உங்கள் செய்திகளை மறைகுறியாக்க முடியும். நீங்கள் காப்புப்பிரதியை உருவாக்கவில்லை அல்லது விசையை இழந்திருந்தால், உங்கள் மறைகுறியாக்கப்பட்ட செய்திகளை துரதிருச்டவசமாக மீட்டெடுக்க முடியாது.

# FluffyChat இல் காப்பு விசையை எவ்வாறு உருவாக்குவது?

FluffyChatஐத் திறந்து, அரட்டை காப்புப்பிரதிக்கு அமைப்புகள் செல்லவும். அங்கு நீங்கள் மறைகுறியாக்கப்பட்ட அரட்டை காப்புப்பிரதியை அமைக்கலாம். மீட்பு விசையைச் சேமிக்கும்படி கேட்கப்படுவீர்கள். முக்கியமானது: இந்த விசையை குறித்துக்கொள்ளவும் அல்லது பாதுகாப்பாக வைக்கவும். அது இல்லாமல், உங்கள் மறைகுறியாக்கப்பட்ட செய்திகளை புதிய சாதனத்தில் மீட்டெடுக்க முடியாது.

# நான் ஒரு பிழையைக் கண்டேன். இப்போது என்ன?

நீங்கள் Github இல் நேரடியாக பிழைகளைப் புகாரளிக்கலாம். மாற்றாக, Messenger இல் (#fluffy-space:matrix.org) ஒரு பொது குழு அரட்டை உள்ளது, அங்கு நீங்கள் கேள்விகளைக் கேட்கலாம் அல்லது கருத்துகளை அனுப்பலாம்.

# Google சேவைகள் இல்லாமல் புச் அறிவிப்புகளை எப்படிப் பெறுவது?

FluffyChat இல் முக்கியமான செய்திகளைத் தவறவிடாதீர்கள் என்பதை உறுதிப்படுத்த புச் அறிவிப்புகள் சிறந்த வழியாகும். Google சேவைகளைப் பயன்படுத்தாத பயனர்களுக்கு (உதாரணமாக Huawei ஃபோன்கள், Amazon Fire டேப்லெட்டுகள் அல்லது தனிப்பயன் ரோம்கள்), புச் அறிவிப்புகளை அமைப்பது சற்று வித்தியாசமானது.

நீங்கள் பயன்பாட்டைத் தொடங்கும் ஒவ்வொரு முறையும் தானாகவே புச் அறிவிப்புகளை அமைக்க FluffyChat முயற்சிக்கிறது. இருப்பினும், உங்கள் சாதனத்தில் Google சேவைகள் இல்லையென்றால், நீங்கள் ஒரு பிழை செய்தியைப் பெறலாம். இந்தச் சந்தர்ப்பத்தில், Ntfy, UP-FCM விநியோகச்தர் (Google) அல்லது உரையாடல்கள் போன்ற UnifiedPush பயன்பாட்டைப் பயன்படுத்துவது தேவை. தானியங்கு புதுப்பிப்பை அனுமதிக்க இங்கே கருத்துத் தெரிவிக்கவும் -->
