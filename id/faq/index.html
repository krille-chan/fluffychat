# Apa itu FluffyChat?

FluffyChat adalah aplikasi sumber terbuka yang ramah pengguna, gratis, dan bebas iklan berdasarkan protokol Matrix dan bekerja serupa dengan Telegram atau WhatsApp.
Dengan FluffyChat, Anda dapat mengobrol dengan teman-teman secara pribadi, membuat obrolan grup, bertukar pesan suara, berbagi gambar dan video, bahkan mengirim lokasi Anda.
Dan bagian terbaiknya:
FluffyChat kompatibel dengan messenger Matrix lainnya!

# FluffyChat ditujukan untuk siapa?

FluffyChat ditujukan untuk siapa saja yang mencari alternatif cepat, ramah pengguna, dan aman dari aplikasi pesan terpusat seperti WhatsApp, Telegram, atau WeChat.

# Siapa yang membuat FluffyChat?

FluffyChat dikembangkan oleh komunitas dan diprakarsai oleh Christian (Krille) Kußowski. Namun ada banyak kontributor, baik kode maupun terjemahan. Anda dapat menemukannya di sini.

# Bagaimana FluffyChat didanai?

FluffyChat adalah proyek nirlaba yang digerakkan oleh sukarelawan dan sepenuhnya didanai oleh donasi.

# Apakah FluffyChat tersedia di perangkat saya?

Ya! Anda dapat menggunakan FluffyChat di ponsel, tablet, dan sebagai versi web di PC Anda. Tersedia di Android (Play Store dan F-Droid), iOS (App Store), sebagai aplikasi web dan aplikasi desktop untuk Linux (Flathub, Snap Store). Bahkan berfungsi di ponsel Linux dengan menggunakan versi Arm64 dari Flatpak/Snap. Anda dapat mengompilasinya sendiri untuk MacOS dan Windows jika mau, tetapi itu belum didukung secara resmi.

# Bagaimana cara membuat akun FluffyChat?

Saat pertama kali membuka aplikasi, Anda akan diminta untuk membuat akun Matrix baru. Pilih server (misalnya, matrix.org), masukkan nama pengguna yang Anda inginkan, dan kata sandi yang aman. Secara opsional, Anda dapat menambahkan alamat email (ini berguna untuk pemulihan kata sandi, tetapi tidak wajib).

# Apa itu Matrix ID?

Matrix ID Anda adalah nama pengguna unik Anda di jaringan, misalnya, @nama:matrix.org. Ini terdiri dari nama Anda dan server tempat akun Anda berada.

# Bagaimana cara masuk ke FluffyChat?

Jika Anda sudah memiliki akun Matrix, cukup masukkan Matrix ID Anda (misalnya, @nama:matrix.org) dan kata sandi. Anda dapat masuk ke server mana pun yang kompatibel dengan Matrix karena FluffyChat bersifat terdesentralisasi.

# Apakah saya memerlukan alamat email untuk FluffyChat?

Tidak, alamat email tidak diperlukan. Namun, disarankan jika Anda ingin mengatur ulang kata sandi atau memverifikasi akun Anda.

# Saya lupa kata sandi saya. Sekarang bagaimana?

Jika Anda memberikan alamat email saat membuat akun, Anda dapat mengatur ulang kata sandi melalui situs web server (misalnya, https://matrix.org). Sayangnya, pemulihan tidak mungkin dilakukan tanpa alamat email. Dalam hal ini, Anda perlu membuat akun baru.

# Saya mengganti perangkat dan sekarang semua pesan saya terenkripsi. Sekarang bagaimana?

Untuk membaca pesan terenkripsi Anda di perangkat baru, Anda perlu memulihkan cadangan yang telah Anda buat sebelumnya. Masukkan kunci cadangan atau kata sandi saat menyiapkan FluffyChat di perangkat baru. Hanya dengan itu pesan Anda dapat didekripsi. Jika Anda belum membuat cadangan atau telah kehilangan kuncinya, pesan terenkripsi Anda sayangnya tidak dapat dipulihkan.

# Bagaimana cara membuat kunci cadangan di FluffyChat?

Buka FluffyChat dan buka pengaturan lalu Cadangan Obrolan. Di sana Anda dapat menyiapkan cadangan obrolan yang terenkripsi. Anda akan diminta untuk menyimpan kunci pemulihan. Penting: Catat kunci ini atau simpan dengan aman. Tanpanya, Anda tidak akan dapat memulihkan pesan terenkripsi di perangkat baru.

# Saya menemukan bug. Sekarang bagaimana?

Anda dapat melaporkan bug langsung di Github. Atau, ada grup obrolan publik di Messenger (#fluffy-space:matrix.org) tempat Anda dapat mengajukan pertanyaan atau mengirimkan masukan.

# Bagaimana cara mendapatkan Notifikasi Push tanpa Layanan Google?

Notifikasi push adalah cara yang bagus untuk memastikan Anda tidak melewatkan pesan penting di FluffyChat. Bagi pengguna yang tidak menggunakan Layanan Google (misalnya Ponsel Huawei, tablet Amazon Fire, atau custom rom), pengaturan notifikasi push sedikit berbeda.

FluffyChat mencoba mengatur notifikasi push secara otomatis setiap kali Anda membuka aplikasi. Namun, jika perangkat Anda tidak memiliki Layanan Google, Anda mungkin menerima pesan kesalahan. Dalam hal ini, Anda perlu menggunakan aplikasi UnifiedPush seperti Ntfy, UP-FCM Distributor (Google), atau Conversations.

Menggunakan Ntfy

Dalam artikel ini, kami akan menunjukkan cara mengonfigurasi notifikasi push di perangkat Android Anda menggunakan aplikasi ntfy.

🏪 Langkah 1: Menginstal dan Menyiapkan ntfy

  1. Jika Anda sudah menginstal FluffyChat tetapi tidak menerima notifikasi push, tutup aplikasi sepenuhnya (dengan menggesernya di tampilan ikhtisar) dan buka PlayStore (atau F-Droid) di perangkat Android Anda.

  2. Cari aplikasi ntfy dan instal.

  3. Setelah menginstal ntfy, buka aplikasi tersebut setidaknya sekali.

  4. Sekarang, kembali ke FluffyChat dan buka aplikasinya.

🔋 Langkah 2: Menonaktifkan Optimasi Baterai untuk ntfy

Untuk memastikan FluffyChat dapat mengirim notifikasi push secara andal, penting untuk menonaktifkan optimasi baterai untuk aplikasi "ntfy".

  1. Buka [pengaturan Android] di perangkat Anda.

  2. Cari "Aplikasi" atau "Manajemen Aplikasi" dan pilih "ntfy" dari daftar aplikasi yang terinstal.

  3. Ketuk "Baterai" dan pilih "Optimasi baterai."

  4. Dalam daftar aplikasi yang dikecualikan dari optimasi baterai, pastikan "ntfy" dipilih.

✅ Langkah 3: Memeriksa Konfigurasi Push di FluffyChat

  1. Buka FluffyChat dan navigasikan ke [Pengaturan].

  2. Pilih "Notifikasi" dari menu.

  3. Di pengaturan notifikasi, Anda akan melihat daftar "Pusher." "Pusher" untuk "ntfy" seharusnya berada di bagian bawah daftar.

🥳 Langkah 4: Menerima Notifikasi Push

Untuk memastikan semuanya telah diatur dengan benar, minta teman untuk mengirim pesan kepada Anda di FluffyChat dan periksa apakah Anda menerima notifikasi push.

Menggunakan Conversations

Jika Anda menggunakan XMPP selain Matrix dan sudah mengonfigurasi Conversations, Anda dapat mengikuti langkah-langkah berikut untuk mendapatkan notifikasi push FluffyChat melaluinya:

  1. Buka Conversations.
  2. Ketuk tiga titik di kanan atas.
  3. Pilih Pengaturan dan gulir ke bawah ke UnifiedPush Distributor.
  4. Ketuk Akun XMPP dan pilih akun XMPP yang ingin Anda gunakan untuk menerima notifikasi.
  5. Buka FluffyChat. Jika tidak menawarkan untuk menerima notifikasi melalui Conversations, periksa pengaturan notifikasi (lihat langkah 3 untuk Ntfy di atas).

Mengikuti langkah-langkah ini seharusnya berhasil mengatur notifikasi push di FluffyChat bagi pengguna tanpa Layanan Google. Jika Anda tertarik untuk meng-host layanan push Anda sendiri, Anda dapat melihat unifiedpush.org.

# Bagaimana cara menggunakan enkripsi ujung ke ujung?

FluffyChat mendukung enkripsi ujung ke ujung. Anda dapat mengaktifkannya per obrolan. Setelah diaktifkan, enkripsi tidak dapat dinonaktifkan lagi demi alasan keamanan. Anda tidak dapat mengaktifkan enkripsi untuk ruang publik karena hal ini tidak masuk akal ketika siapa saja bisa bergabung dan meninggalkan ruangan tersebut. Setelah Anda mengaktifkan enkripsi, server tidak lagi dapat membaca isi pesan Anda. Enkripsi akan berlaku pada semua pesan teks dan file.

Untuk melindungi diri dari serangan man-in-the-middle, Anda harus melihat daftar perangkat. Anda dapat melakukan ini di pengaturan enkripsi sebuah obrolan. Apakah ada perangkat mencurigakan yang Anda atau mitra obrolan Anda tidak kenali? Anda dapat memblokirnya dengan menggeser tombol ke kiri.

Warna tombol menunjukkan status keamanan. Oranye berarti perangkat ini belum diverifikasi. Abu-abu berarti diblokir dan tidak akan menerima kunci enkripsi. Ini berarti perangkat tersebut tidak akan dapat membaca pesan Anda lagi! Hijau berarti perangkat ini telah diverifikasi.

Untuk memverifikasi perangkat pengguna lain, mulailah verifikasi dalam obrolan langsung. Untuk ini Anda memiliki tombol "Mulai verifikasi" di pengaturan enkripsi obrolan langsung. Pastikan bahwa Anda berada di samping orang tersebut atau terhubung melalui telepon atau saluran aman lainnya. Ketika Anda telah memulai verifikasi, Anda berdua akan melihat sekumpulan emoji di perangkat Anda. Bandingkan dan ketuk "Cocok". Tunggu sebentar dan kemudian Anda akan melihat semua perangkat menjadi hijau.

# Bagaimana cara melakukan fork FluffyChat?

Daftar Isi

  • 1. Lisensi
  • 2. Nonaktifkan enkripsi ujung ke ujung!
  • 3. Tetap perbarui!
  • 4. Dukungan Resmi

1. Lisensi

FluffyChat dilisensikan di bawah AGPL. Baca lisensinya (https://github.com/krille-chan/fluffychat/blob/main/LICENSE) dan pastikan bahwa fork Anda bersifat sumber terbuka di bawah lisensi yang sama dan Anda memenuhi semua persyaratannya. Mungkin Anda sebaiknya mempertimbangkan untuk menghubungi pengacara sebelum Anda mempublikasikan fork Anda.

2. Nonaktifkan enkripsi ujung ke ujung!

Karena peraturan ekspor AS, Anda tidak diizinkan untuk mempublikasikan aplikasi Anda di toko atau di server AS mana pun sebelum Anda menghapus semua hal yang berkaitan dengan enkripsi atau memenuhi peraturan tersebut.

Pelajari lebih lanjut: https://www.bis.doc.gov/index.php/policy-guidance/encryption

Jika Anda membutuhkan bantuan dari kami dalam menggunakan E2EE di fork Anda, baca lebih lanjut di bawah topik "Dukungan Resmi".

3. Tetap perbarui!

FluffyChat mengandung hal-hal yang berkaitan dengan keamanan. Jika kami menemukan bug keamanan, kami akan berusaha memperbaikinya sesegera mungkin dan merilisnya dengan versi baru. Namun ini berarti fork Anda sudah usang dan merupakan risiko keamanan. Anda tidak bisa terjaga 24 jam sehari sehingga Anda harus memutuskan bagaimana cara melindungi pengguna Anda dengan memilih salah satu metode berikut:

  1. Buat fork Anda seminimal mungkin dan aktifkan pencerminan repositori. Siapkan CI yang mempublikasikan versi baru secara otomatis jika FluffyChat mempublikasikan perbaikan bug.
  2. Tidak pernah tidur dan membayar tim besar di mana setidaknya satu orang tidak pernah tidur.
  3. Hubungi famedly.com untuk membeli dukungan resmi.

4. Dukungan Resmi

FluffyChat gratis dalam artian kebebasan berbicara, bukan bir gratis! Harap hubungi perusahaan saya famedly.com untuk penawaran dan dukungan resmi, dan ingat bahwa dibutuhkan banyak pekerjaan dan waktu untuk memelihara FluffyChat atau Famedly Matrix SDK. Jadi kami tidak dapat memberikan dukungan secara gratis. Harap perkirakan sekitar $1 per bulan per pengguna dari fork Anda.

# Bagaimana cara menemukan pengguna lain?

Ada beberapa cara untuk menemukan pengguna lain di FluffyChat.

  1. Cari berdasarkan nama tampilan: Anda dapat memasukkan nama tampilan seseorang di kolom pencarian. Ini hanya berfungsi jika:
    • Anda sudah pernah berbagi obrolan dengan orang tersebut,
    • Mereka berada di ruang publik yang dikenal server Anda,
    • Atau server Anda mengizinkan pencarian pengguna lain.
  2. Masukkan Matrix ID secara langsung: Jika Anda mengetahui Matrix ID (misalnya, @krille:janian.de), Anda dapat memasukkannya langsung di kolom pencarian. Cara kerjanya mirip dengan alamat email: @namapengguna:namaserver.
  3. Pindai kode QR: Anda dapat memindai kode QR orang lain atau menampilkan kode QR Anda sendiri untuk dipindai.
  4. Bagikan tautan undangan: Siapa pun dapat berbagi tautan undangan ke sebuah ruang atau obrolan. Mengklik tautan tersebut akan menghubungkan Anda secara langsung.

# Bagaimana cara mendapatkan stiker?

Anda dapat menambahkan paket stiker dari internet. Misalnya volpeon.ink melakukan pekerjaan yang luar biasa:

https://volpeon.ink/emojis/

Ikuti langkah-langkah berikut:

  1. Unduh salah satu paket sebagai file zip

  2. Buka FluffyChat

  3. Pergi ke "Pengaturan" > "Obrolan" > "Emote dan stiker khusus"

  4. Ketuk menu tiga titik di kanan atas dan ketuk "Impor dari file zip"

  5. Pilih file zip dan jika mau, edit nama stikernya

Anda juga dapat menemukan paket stiker yang sudah dikonfigurasi di Matrix. Saya merekomendasikan Koleksi Stiker Nico:

https://matrix.to/#/#nico's-stickers:neko.dev

Koleksi stiker untuk FluffyChat sayangnya belum memiliki pilihan yang banyak:

https://matrix.to/#/#fluffychat-stickers:janian.de

  1. Bergabunglah ke ruang publik

  2. Buka menu tiga titik di kanan atas dan ketuk "Pengaturan Emote"

  3. Pilih paket stiker di ruang yang Anda sukai

  4. Aktifkan "Aktifkan paket emote secara global"

Selamat menikmati stiker Anda! :-)
